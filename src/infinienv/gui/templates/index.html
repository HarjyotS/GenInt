InfiniEnv

Infinite environment generation via an agent harness — type a prompt, pick your settings, watch it build.

Trades away the validator-guaranteed solvability check every other run has, in exchange for mechanics the fixed vocabulary can't express (see CLAUDE.md section 11). Ignores provider/--no-fallback below. Assets still applies -- the sandbox agent resolves real sprites the same way any other run does. If the first attempt fails an independent outer sanity check, the same agent gets the concrete failure and repairs its own work, up to max repair attempts.
Applies in --sandbox mode too.
mock needs no API key. openai_agents is the default full runtime.
